dns

control_service

시스템의 중앙 허브. ROS2 노드로 로봇과 통신하고, TCP 서버(8080)로 Admin UI를, REST API(8081)로 고객 웹앱을 연결합니다.

language

customer_web

고객이 사용하는 모바일 웹앱. Flask + SocketIO로 구성되어 QR 스캔 후 로그인·쇼핑·결제 플로우를 제공합니다. (포트 8501)

model_training

ai_service

Docker 컨테이너로 실행되는 AI 추론 서버. 커스텀 YOLOv8 모델(TCP:5005)과 LLM 가이드 서버(REST:8000)를 제공합니다.

storage

control_db

PostgreSQL 17 DB 스키마 및 시드 데이터. 고객·상품·주문 정보와 ReID 특징벡터(pgvector)를 저장합니다.

route

shoppinkki_rmf

Open-RMF Fleet Adapter. 여러 로봇을 중앙에서 조율하는 다중 로봇 관리 레이어입니다.
