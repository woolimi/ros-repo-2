추종: IoU 트래커

핵심 아이디어

IoU(Intersection over Union)는 두 bbox의 겹치는 비율입니다. ByteTracker 내부에서도 사용되며, Pi 로컬에서도 경량 IoU 트래커로 ID를 안정화합니다.

공식

IoU = 교집합 넓이 / 합집합 넓이
교집합
두 bbox가 겹치는 영역
합집합
두 bbox의 전체 영역
범위
0 (겹침 없음) ~ 1 (완전 일치)

주요 파라미터

min_iou = 0.3
이 이상 겹쳐야 동일 객체로 매칭
max_age = 10
연속 미감지 시 트랙 삭제 (프레임)

매 프레임 동작

1 기존 트랙 × 신규 bbox 간 모든 IoU 계산
2 가장 높은 IoU 쌍부터 탐욕적(greedy)으로 매칭
3 매칭 → 기존 ID 유지 / 미매칭 → 신규 ID 부여
