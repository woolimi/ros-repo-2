integration_instructions

shoppinkki_interfaces

Python Protocol로 하드웨어·Nav2 의존성을 추상화한 계층. 테스트 시 Mock으로 대체할 수 있어 실제 로봇 없이도 단위 테스트가 가능합니다.

hub

shoppinkki_core

로봇의 두뇌. 상태머신이 현재 상태를 판단하면 bt_runner가 그에 맞는 Behavior Tree를 실행합니다. 매장 경계 감시와 LCD·LED·버저 등 하드웨어 제어도 여기서 담당합니다.

near_me

shoppinkki_nav

Nav2 기반 자율주행 담당. 추종·가이드·복귀 등 각 상태별 Behavior Tree를 포함합니다.

visibility

shoppinkki_perception

카메라 기반 인식 패키지. YOLOv8 감지 결과 수신·ReID 특징 추출·HSV 매칭을 담당합니다.
