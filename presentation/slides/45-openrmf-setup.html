가이드: Open-RMF 사용을 위한 준비

1

Nav Graph 정의

매장 레이아웃에 맞게 웨이포인트와 경로를 shop_nav_graph.yaml로 작성

2

Fleet Adapter 구현

EasyFullControl API로 로봇 등록, navigate 콜백에서 Nav2 goal 전달

3

위치·상태 피드백

로봇의 실시간 위치·배터리를 RMF에 주기적으로 보고

4

RMF Core 서버 실행

rmf_traffic_schedule · rmf_task 노드를 launch로 실행

code fleet_adapter.py 
# 1) Fleet 설정
config = EasyFullControl.Configuration(
    fleet_name="shoppinkki",
    nav_graph_path="shop_nav_graph.yaml"
)

# 2) 로봇 등록
fleet.add_robot(
    robot_name="pinky_54",
    navigate=on_navigate,
    stop=on_stop
)

# 3) 위치 보고
robot_handle.update(position, battery_soc)
