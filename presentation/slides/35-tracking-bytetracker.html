ByteTracker란?

YOLO bbox에 일관된 track_id를 부여하는 다중 객체 추적 알고리즘.
AI 서버에서 model.track(persist=True) 한 줄로 활성화됩니다.

yolo_server.py
# ByteTracker 적용 (persist=True)
results = model.track(
    img,
    persist=True,   # 프레임 간 트랙 유지
    conf=confidence,
    imgsz=640
)

# track_id 추출
track_id = int(det.boxes.id[0]) if det.boxes.id is not None else -1

Hungarian Algorithm — 최적 매칭

예측된 트랙들과 새로 감지된 bbox들을 전체 비용이 최소가 되도록 최적으로 매칭합니다.
