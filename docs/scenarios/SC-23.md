# SC-23: 대기 중 통행자 감지 → 회피 이동

| 항목 | 내용 |
|---|---|
| **상태 전환** | `WAITING` 내부 (SM 변경 없음) |
| **트리거** | BT 3 — RPLiDAR 근접 통행자 감지 → Nav2 소폭 이동 |
| **우선순위** | LOW |
| **연관** | UR-13, SR-43 |

---

## 사전 조건

- SM = `WAITING`
- BT 3 동작 중 (10 Hz 루프)

## 시험 절차

1. **SM = `WAITING` 상태 진입**

2. **통행자 시뮬레이션: 로봇 전방 근접 거리에 장애물 배치**
   - 실 환경: 사람 또는 물체를 `MIN_DIST(=0.25m)` 이내로 접근
   - Gazebo: 스폰 명령으로 장애물 배치

3. **LiDAR 수신 확인**
   ```bash
   ros2 topic echo /scan
   ```
   - 전방 반원 내 `MIN_DIST` 이하 거리 확인

4. **BT 3 회피 동작 확인**
   - 회피 방향 결정 (좌우 여유 공간 비교)
   - Nav2 Goal 전송 (현 위치 기준 ~0.3 m 측방)
   - 이동 완료 후 정지 복귀

5. **장애물 제거 후 SM 유지 확인**
   - SM = `WAITING` 그대로

## 예상 결과

- SM 상태 변경 없음 (`WAITING` 유지)
- 로봇이 소폭 측방 이동 후 정지 복귀
- [앱] 뱃지 "대기 중" 유지
- `/cmd_vel` 에 이동 명령 후 zero 복귀 확인
